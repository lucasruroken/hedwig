# Hedwig

Hedwig is an API wrapper for [TripAdvisor's Content API](https://developer-tripadvisor.com/content-api/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hedwig'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hedwig
